- **中文无敌**：写作、总结、翻译，理解力和表达力都是顶级
- **代码能打**：推理、编程、智能体能力达到开源 SOTA 水平
- **访问稳定**：国内直连，不用翻墙，响应速度快
- **价格真香**：Coding Plan 套餐 **49 元/月起**，比按量付费省太多

### 🎯 GLM-5 核心优势

| 能力 | 说明 |
|------|------|
| 深度推理 | 复杂逻辑分析、多步骤任务规划 |
| 代码生成 | 语法正确、风格统一、可直接运行 |
| 智能体协作 | 支持 Agent 自动调用工具、完成多轮任务 |
| 中文表达 | 写作润色、文档总结、技术翻译 |

**Pro 套餐额外解锁**（强烈推荐）：

## 跟我做

### 第 1 步：注册并获取 API Key

推荐通过以下链接注册/登录：

- **教程站专属链接（优惠 10%）**：[https://www.bigmodel.cn/glm-coding?ic=PNSI6JU6FP](https://www.bigmodel.cn/glm-coding?ic=PNSI6JU6FP)

> **提示**：
> - 建议选 Pro 套餐，能用 GLM-5 还能解锁联网搜索等高级能力
> - 拿到 Key 后先保存好（不要发群、不要提交到 GitHub）

---

### 第 2 步：在 OpenCode 中连接智谱

启动 OpenCode：

```bash
opencode
```

输入：

```
/connect
```

在提供商列表里搜索并选择：

- `Zhipu AI Coding Plan`

然后粘贴你的 API Key。

---

### 第 3 步：选择 GLM-5 模型

输入：

```
/models
```

选择 `glm-5` 即可。

---

### 第 4 步：发送第一句话（验证成功）

```
你好，请介绍一下你自己
```

能收到回复就说明连接成功。

## 踩坑提醒

| 现象 | 原因 | 解决 |
|-----|------|------|
| 没有 `Zhipu AI Coding Plan` 选项 | OpenCode 版本太旧 | 升级：`opencode upgrade` |
| `API key invalid` | Key 格式错误 | 确认是完整的 Key（格式类似 `xxx.xxx`） |
| 响应慢或超时 | 网络波动 | 检查网络，或换个时间段 |
