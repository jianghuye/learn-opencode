# v1.1.21：官方 Copilot 插件来了！

**发布日期**：2026年01月15日

---

## 一句话总结

这次更新带来了官方的 GitHub Copilot 插件支持，并为 Windows 用户新增了 Chocolatey 和 Scoop 升级方式。

---

## 新功能：官方 Copilot 插件

### 这次更新了什么？

OpenCode 现在内置了官方的 GitHub Copilot 插件支持，让你可以更顺滑地使用 Copilot 的能力。

### Copilot 插件是什么？

GitHub Copilot 是一个强大的 AI 编程助手。通过这个官方插件，OpenCode 可以直接调用 Copilot 的能力来辅助你写代码，不需要复杂的配置。

### 如何使用？

**快速体验**：

1. 打开 OpenCode。
2. 在设置或插件管理中查找 Copilot 相关配置（通常会自动检测或提示）。
3. 如果遇到认证问题，现在的错误提示会引导你重新认证。

---

## 新功能：Windows 包管理器升级支持

### 这次更新了什么？

Windows 用户现在可以使用 `choco` (Chocolatey) 或 `scoop` 来升级 OpenCode 了。

### 是什么？

Chocolatey 和 Scoop 是 Windows 上常用的软件包管理工具，类似 macOS 的 Homebrew。

### 如何使用？

如果你是通过这两个工具安装的 OpenCode，现在可以用标准命令升级：

**Chocolatey**:
```powershell
choco upgrade opencode
```

**Scoop**:
```powershell
scoop update opencode
```
