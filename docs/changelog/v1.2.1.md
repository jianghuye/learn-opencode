# v1.2.1：跨目录查看会话

**发布日期**：2026年2月14日

---

## 一句话总结

这次更新让你可以在**任意目录查看所有项目的会话记录**，同时优化了网络搜索的缓存行为。

---

## 新功能：跨目录查看所有会话

### 之前的问题

你可能在多个项目目录下使用 OpenCode，每个目录有自己的会话。之前想查看某个项目的会话，必须先 `cd` 到那个目录。

### 现在的情况

**从任意目录都能看到所有项目的会话**。不管你在哪个目录，OpenCode 都会列出你所有项目的会话记录。

### 这有什么好处？

- 不用频繁切换目录
- 快速找到之前的对话
- 管理多项目更方便

---

## 优化：网络搜索更省流量

### 改了什么？

微调了 `websearch` 工具的日期信息描述，避免每次搜索都绕过缓存。

### 对你有什么影响？

- 搜索响应更快（有缓存时）
- 减少 API 调用
- 整体更省流量

---

## 要不要升级？

| 情况 | 建议 |
|------|------|
| 经常切换多个项目 | ✅ 升级 |
| 想要更好的会话管理 | ✅ 升级 |
| 用得好好的 | 可以等下次 |

**升级命令**：

```bash
opencode upgrade
```
