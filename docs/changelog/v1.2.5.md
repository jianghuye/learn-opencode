**发布日期**：2026年2月15日

这次更新让**桌面应用更好用了**，修复了文件打开问题，添加了新字体和快捷键。

## 问题修复：文件树打开新文件

### 之前的问题

在桌面应用中，从文件树打开新文件时，有时候无法正常查看。

### 现在的情况

这个问题已经修复，现在可以流畅地打开和切换文件了。

## 新功能：Shift+Tab 快捷键

### 这次更新了什么？

在桌面应用中添加了 `Shift+Tab` 键盘快捷键。

### 用在哪？

`Shift+Tab` 通常用于**反向导航**（与 Tab 相反），例如：
- 从当前输入框回到上一个输入框
- 反向切换可交互元素

---

## 新功能：GeistMono Nerd Font

### 这次更新了什么？

在桌面应用的字体选项中新增了 **GeistMono Nerd Font**。

### GeistMono 是什么？

GeistMono 是一套现代、干净的开源等宽字体，特别适合写代码：
- 清晰易读
- 支持多种编程语言的特殊字符
- 视觉风格简洁

### Nerd Font 是什么？

Nerd Font 是在原字体基础上添加了**大量图标**的版本，包括：
- Git 分支图标
- 编程语言图标
- 文件类型图标
- 等等

**简单说**：既美观又实用！

### 如何切换字体？

1. 打开桌面应用
2. 进入 **设置**
3. 找到 **字体设置**
4. 在等宽字体选项中选择 **GeistMono Nerd Font**
5. 应用并享受新字体

## 问题修复：窗口焦点

### 之前的问题

更新或重启后，桌面应用的窗口可能失去焦点，需要手动点击才能继续输入。

### 现在的情况

更新和重启后会自动聚焦窗口，你可以直接继续使用。

---

## 问题修复：提示历史导航

### 之前的问题

在使用上/下箭头查看提示历史时，有时候会在输入框中间的任意位置切换历史，体验不好。

### 现在的情况

现在只有在**输入边界**（光标在最前或最后）才会切换历史记录，中间位置不会误触发。
